Welford's Algorithm

Welford's algorithm is an efficient method for computing the mean and variance of a dataset in a single pass. Unlike the naive method, it avoids numerical instability caused by subtracting large numbers, making it well-suited for streaming data or cases where the dataset is processed incrementally.

The mean is updated using the following formula:

\[ m_t = m_{t-1} + \frac{x_t - m_{t-1}}{t} \]

where \(m_{t-1}\) is the mean calculated from the previous \(t-1\) data points, when a new data point, \(x_t\), arrives at time \(t\).

The formula for updating the variance is:

\[ s_t^2 = s_{t-1}^2 + (x_t - m_{t-1})(x_t - m_t) \]

where \(s_{t-1}^2\) is the variance from the previous data points.

The unbiased sample variance is:

\[ \hat{\sigma}^2 = \frac{s_t^2}{t-1} \]
